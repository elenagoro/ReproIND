# ReproIND Threshold Report

This repository contains an RMarkdown script that calculates the P90 thresholds for embryo malformation and affected females using bootstrapped resampling.

## Contents

- `ReproIND_Thresholds_Report.Rmd` – main analysis script
- `malf.csv` – input data file (or example)
- Output plots and CSVs

## How to Use

1. Open the RMarkdown file in RStudio
2. Knit to HTML
3. Modify the input file as needed
